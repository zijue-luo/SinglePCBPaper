Trap Electronics Experiment — Load → Wait → Extract

Detector: MCP (efficiency factor d considered). Records: N = 500

Schema & Definitions

  • loading_signal, lost_signal, trapped_signal
  • ratio_signal = trapped/loading (checked)
  • ratio_lost = lost/loading (checked)
  • Ex, Ey, Ez; time_cost, time_accum
  • Predicted loading_signal; Detrended_loading_signal

Sanity Checks

Mean absolute error for provided ratios vs recomputed: ratio_signal = 6.66e-12, ratio_lost = 2.49e-11.

Global Linear Models

trapped ~ a·loading + b·lost + c → R² = 0.473, a = -0.001569, b = 3.149674, c = 10.007.

trapped ~ a·Detrended_loading + b·lost + c → R² = 0.467, a = -22.676, b = 3.117, c = 6.425.

Regime Behavior

High-loading regime (top 20% by loading): trapped ≈ k·lost + c with k = 0.556, c = -6.673, R² = 0.969.

Time & Drift

Corr(Detrended_loading_signal, Ez) = 0.803. Residuals show negligible correlation with time_accum overall.
